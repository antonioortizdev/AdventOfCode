## Debugging
Open 'launch.json' and change "program" to the path of the file you want to debug.
```json
{
    "version": "0.2.0",
    "configurations": [
        {
            "name": "Python: Current File",
            "type": "python",
            "request": "launch",
            "program": "${workspaceFolder}/2023/1/1/main.py",
            "console": "integratedTerminal"
        }
    ]
}
```
